## ✨ Overview

**FivemUCP-Demo** is an example of a modern FiveM User Control Panel (UCP).

The goal of this project is to provide a clean, responsive and immersive dashboard experience for FiveM communities.

Inspired by professional roleplay platforms, the UCP focuses on:

- 🎭 Character management
- 📝 Whitelist applications
- 👤 Player profiles
- 📊 Server statistics
- 🔐 Secure authentication
- ⚡ Fast modern UI

- View player information
- Character overview
- Account details
- Statistics dashboard

## 📝 Whitelist System

Complete application workflow:

✅ Create applications  
✅ Custom questions  
✅ Admin review queue  
✅ Approve / deny applications  
✅ Application history

Secure login system:

- NextAuth authentication

- Protected dashboard routes
- Database-backed users
